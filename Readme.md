# 🌾 Flowzint

> **Empowering Farmers Through AI, Smart Marketplace & Intelligent Logistics**

Flowzint is an AI-powered smart agriculture platform that connects farmers, buyers, and logistics providers into a single digital ecosystem. The platform enables transparent crop trading, real-time auctions, AI-powered farming assistance, logistics tracking, and credit services to modernize the agricultural supply chain.

---

## ✨ Key Features

- 🌾 Direct Farmer-to-Buyer Marketplace
- ⚖️ Live Crop Auctions
- 🤖 KrishiSarthi AI Assistant
- 🚚 Smart Logistics & Delivery Tracking
- 📦 Order & Inventory Management
- 💳 Credit Scoring & Loan Support
- 📊 Farmer Analytics Dashboard
- 🌍 Multilingual Support (English, Hindi & Marathi)

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/Flowzint.git
cd Flowzint
```

### 2. Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
GROQ_API_KEY=your_groq_api_key
```

Run the backend:

```bash
npm run dev
```

---

### 3. Frontend Setup

```bash
cd frontend
npm install
```

Create a `.env.local` file:

```env
NEXT_PUBLIC_API_URL=http://localhost:5000/api
```

Run the frontend:

```bash
npm run dev
```

Visit:

```
http://localhost:3000
```
